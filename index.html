openui5-cards Goal

The goal is to provide Google Now like cards for the OpenUI5 framework. The focus will be on mobile experience.

Overview

The openui5-cards are implemented as custom components to OpenUI5

The components are currently:

  • open.m.CardContainer
  • open.m.Card
  • open.m.CardActions

  • Address (static map image fetched from Google Maps API based on this)
